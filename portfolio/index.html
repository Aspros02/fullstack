PROYECTOS

GACE: GESTOR DE TORNEOS DE VIDEOJUEGOS

  • Descripción: Web para la gestión de torneos de videojuegos. Proyecto basado en MVC, donde las empresas pueden registrarse, agregar juegos de su compañía y publicar torneos. Los jugadores pueden registrarse a dichos torneos, obteniendo un justificante en formato pdf. De forma adicional, la web presenta una serie de cápsulas informativas sobre el mundo de los videojuegos.

  • Tecnologías: HTML, CSS, framework Bootstrap y JavaScript para la interfaz y funcionalidades. PHP para la comunicación con la base de datos relacional SQL, creada en Workbench. Creación de un servidor local mediante XAMPP.

Link a Github

APP MÓVIL: LISTA DE LA COMPRA COMPARTIDA

  • Descripción: Aplicación móvil para la creación de listas de la compra. Permite la inserción y el borrado de elementos de la lista, almacenados en una base de datos en la nube.

  • Tecnologías: HTML, CSS y JavaScript. Mediante JS se ha inicializado una aplicación en Firebase para crear una base de datos accesible desde diferentes dispositivos móviles.

Link a Github

USO DE API'S: POKEDEX

  • Descripción: Uso de la API oficial de Pokemon para la generación de una Pokedex con sistemas de filtro. Se puede controlar el número de salidas disponibles en la interfaz.

  • Tecnologías: HTML, CSS y JavaScript para la realización de peticiones a una URL determinada.

Link a Github

EXTENSIÓN GOOGLE CHROME: ALMACÉN DE LINKS

  • Descripción: Desarrollo de una extensión para Google Chrome que permite copiar y almacenar links fácilmente desde la propia interfaz del buscador.
